Historien om en Sjuk Onkel 


Han ble borte av tunge
medisiner 

det ble og familien som 
sønn og søster og kone

dem ble sjuke og 
frisk var bare preseten
som tok farvell 


dette er korteste 
levesettet av onkel
på tunge medisiner 
rett før møtt  presten
og ikke kunne tale for seg
.
